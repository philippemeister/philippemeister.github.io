GVL Design System

Executive Summary

Global Veterinary Link (GVL) is a animal heath software company. As a user experience intern, my primary project was to document existing practices and develop new pracitices associated with a design system. This work included auditing existing design components, building working relatonships accross the company, designing new components, and implimenting a web-based system (we used Adobe XD, Zeroheight, and Storybook). This work didn't begin with me and it will not stop with me, but I completed some tangible steps to advance the design system at GVL. I also assessed APIs for international addresses and prototyped designs that were implimented in the system.
